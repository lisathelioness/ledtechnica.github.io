W3 Total Cache and Amazon CloudFront

8 months ago

Recently I installed W3 Total Cache plugin, which I heard good things about, in one of my blogs that is hosted on Digital Ocean Ubuntu droplet. Once I went in to configure the plug-in, however, the Amazon CloudFront option was greyed out under the CDN settings.

In case someone else runs into the same issue, the fix was to install the php5-curl package with the following command:

sudo apt-get install php5-curl
